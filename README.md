# Kamran Reza Personal Website

Static GitHub Pages site for the `KamranReza` account.

## Launch on GitHub Pages

1. Create a public GitHub repository named `KamranReza.github.io`.
2. Upload every file in this folder to that repository.
3. In GitHub, open Settings, then Pages.
4. Set the source to `Deploy from a branch`, choose `main`, and save.
5. The site will publish at `https://KamranReza.github.io`.

## Contact Form

The form uses FormSubmit and sends messages to `kreza010@ucr.edu`. On the first real submission, FormSubmit may send a confirmation email to activate the address.
